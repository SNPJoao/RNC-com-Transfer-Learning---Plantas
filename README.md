RNCs com transfer learning em um Dataset de plantas

O código utiliza 2 modelos diferentes para realizar o transfer learning: O MobileNet_V2 e o DenseNet121.
Foi utilizado o pytorch para a utilização das IAs e seu treinamento.

O dataset pode ser encontrado nos sites:
    -https://www.kaggle.com/code/atharvaingle/plant-disease-classification-resnet-99-2
    -https://github.com/spMohanty/PlantVillage-Dataset

Ao baixar o dataset do github, basta descompactar, obter o caminho e alterar no código "duasRNCs.py".
O código já possui o caminho até os datasets, basta obter o código até o diretório "PlantVillage-Dataset-master".
